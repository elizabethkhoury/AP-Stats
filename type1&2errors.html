Type I and Type II Errors

Type I Error Type II Error

Type I Error

We reject the null hypothesis (H₀) when H₀ is actually true.
Therefore, we believe there is significant evidence to suggest that H₀ is not true, when it actually is.

Probability:

The probability of making a Type I error is called the alpha-level (α).

Type II Error

We fail to reject the null hypothesis (H₀) when the Hₐ is true.
Therefore, we do not believe there is significant evidence to suggest that Hₐ is true, when it actually is.

Probability:

The probability of making a Type II error is 1 - power.
